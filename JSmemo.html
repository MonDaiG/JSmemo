toc inputのtype

inputタグのtype属性には以下のような種類があります。

<input type="button">
クリック可能なボタン
<input type="checkbox">
チェックのオンオフを切り替えられるボタン
<input type="color">
色を選択させるウィンドウを起動する
<input type="date">

年月日を選択するためのフォーム
<input type="datetime-local">

年月日に加え、時間も選択させるフォーム
<input type="datetime">

日付と時刻に加えてタイムゾーンを入力する
ただし、もうそろそろなくなる仕様
<input type="email">

メールアドレスを入力させるためのフォーム
<input type="file">

ファイル選択のウィンドウを起動する
<input type="hidden">
何も表示しないフォーム。隠しフォーム？
<input type="image">
画像を使った送信ボタン
<input type="month">

年と月まで指定するフォーム
<input type="number">

整数値の入力させるためのフォーム。上下の矢印で選択することもできる。
<input type="password">

パスワード入力用のフォーム。入力した文字が表示されない
<input type="radio">
spam? spam. spam!
ラジオボタンは複数並べてこそチェックボックスとの違いがわかる。
ラジオボタンのグループ化する(複数選択を不可にする)ためにはname属性を揃える必要があります。
<input type="range">
0~100
0~5
スライダー。min,max属性で最小値、最大値を設定し、stepで刻み幅を設定する
スライダー自体の幅はcssのwidthプロパティで設定できる
<input type="reset">

リセットボタン。クリックすると入力フォームの値をリセットするらしいが？
<input type="search">

検索用窓。textフォームと機能的には同じらしい
<input type="submit">

送信ボタン。フォームに入力された内容をサーバに送信しようとするよ！
<input type="tel">

電話番号入力用フォーム。
スマホで入力しようとすると、数字入力用のキーパッドを起動する点がtextフォームと違うよ！
<input type="text">

シンプルなテキスト入力用フォーム。
value属性で初期値、minlength,maxlengthで最小文字数、最大文字数を設定できるよ！
size属性は入力フォームの長さを設定できるよ！maxlengthより少し長めがオススメ！
<input type="time">

時間入力用フォーム。
<input type="url">

URL入力用フォーム。送信前に有効な値かどうかチェックするよ！
:validプロパティで有効時の書式、:invalidプロパティで無効時の書式を設定できるよ！
<input type="week">

年と週を設定するフォーム。使える……のか……？
